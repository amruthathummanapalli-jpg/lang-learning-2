 Language Learning App

 Description

This project is a simple Language Learning Application built using Java.
It helps users learn basic words in different languages.

Languages

* English
* Hindi
* Spanish

 Features

* Language selection
* Displays basic vocabulary
* Simple Java program for beginners

Technologies Used

* Java
* Scanner Class
* Switch Case

 How to Run

1. Compile the program:
   javac Main.java
2. Run the program:
   java Main
